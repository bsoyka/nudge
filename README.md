# Nudge

This project, made for Minnehack 2026, is a habit tracker based on the idea of friends verifying you've completed your daily habits.

Nudge is built on React, Next.js, Tailwind CSS, and Firebase. It can be run locally with npm:

```bash
npm run dev
```

Open [localhost:3000](http://localhost:3000) with your browser to see the result.
